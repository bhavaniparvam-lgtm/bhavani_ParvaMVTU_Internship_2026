2. User-defined functions - there are 3 types

1. function declaration

        function fun_name(parameters){
            //code

        fun_name(args);

2. function expression

        const fun_name = function(parameters){
            //code
        }
        fun_name(args);

3.Arrow fuunction

        const fun_name = (parameters) => {
            //code
        }
        fun_name(args)
