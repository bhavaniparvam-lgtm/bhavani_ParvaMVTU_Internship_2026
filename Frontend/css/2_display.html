html element types

1. block level elements takes full width of the container and starts on a new line ex: div, p, h1-h6

2. inline elements only takes the width of their content and do not start on a new line ex: span, a, img, strong, em

1.display property in CSS controls how an element is displayed on a webpage. 

        selectors {
            display: value;
        }

display : block - makes the element a block-level element, taking up the full width of its container and starting on a new line.
