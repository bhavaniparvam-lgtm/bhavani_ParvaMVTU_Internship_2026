Positioning in CSS

Positioning in CSS refers to how elements are positioned within the document flow. 

    1. static: This is the default position for all elements. Elements are positioned according to the normal flow of the document.
    2. relative: Elements are positioned relative to their normal position. You can use the top, right, bottom, and left properties to adjust the position of the element.
    3. absolute: Elements are positioned relative to the nearest positioned ancestor (an ancestor with a position other than static). If there is no such ancestor, the element is positioned relative to the initial containing block (usually the viewport).
    4. fixed: Elements are positioned relative to the viewport, which means they stay in the same position even when the page is scrolled. You can use the top, right, bottom, and left properties to adjust the position of the element.
    5. sticky: Elements are positioned based on the user's scroll position. A sticky element toggles between relative and fixed, depending on the scroll position. It is treated as relative until it crosses a specified threshold, at which point it becomes fixed.
