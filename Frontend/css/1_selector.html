Selectors in CSS

Selectors in CSS are used to target and style specific HTML elements on a webpage. They allow you to apply styles to elements based on their type, class, ID, attributes, and more. Here are some common types of selectors in CSS:

1. **Element Selector**: Targets all elements of a specific type. For example, `p` will target all `

` elements. 2. **Class Selector**: Targets elements with a specific class attribute. For example, `.my-class` will target all elements with the class "my-class".

3. **ID Selector**: Targets a single element with a specific ID attribute. For example, `#my-id` will target the element with the ID "my-id".

4. **Universal Selector**: Targets all elements on the page. It is represented by an asterisk (*). For example, `*` will target all elements.
